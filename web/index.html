Apple Health Setup

Sync weight (and other Health metrics) from iPhone via the Health Auto Export app:

  1. Install Health Auto Export on your iPhone.
  2. Open the app → Automations → REST API.
  3. Add a new automation, paste this URL:
  4. Pick metrics: Body Mass, Height, Resting Heart Rate, VO2 Max.
  5. Save. Run ./run.sh dash --host 0.0.0.0 on this Mac so iPhone can reach it.

Or for an instant pull, install the WhoopPullWeight shortcut and tap "Pull from iPhone".
